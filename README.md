# Computer Equipment Web Shop

## Technologies:
* **Frontend**: React.js with Bootstrap5
* **Backend**: Node.js with Express framework
* **Database**: MySQL
* **Authentication & Security**: JSON Web Tokens (JWT) and BCrypt for password hashing
* **Payments**: Stripe API integration

## App Description:
This project is a modern e-commerce solution for selling computer components, developed to address common shortcomings in local online stores, such as the lack of user reviews and concerns regarding data security. 
The application utilizes an asynchronous Node.js backend for efficient request handling and a React frontend for an optimized, fluid user experience.

## Functionalities:

### User Functionalities:
* **Registration & Login**: Secure authentication with encrypted passwords using BCrypt algorithm and session management via JWT.
* **Product Search**: Efficient item lookup based on keywords.
* **Shopping Cart**: Full cart management â€“ add products, modify quantities, and automatic total price calculation.
* **Checkout & Delivery**: User-friendly shipping information entry and order summary before final payment.
* **Online Payment**: Integrated secure credit card processing via the Stripe platform.
* **Reviews & Ratings**: Users can provide feedback and rate products (1-5 stars) after purchase.
* **Profile Management**: Access to personal order history and profile detail updates.

### Admin Functionalities:
* **Product Management (CRUD)**: Complete control over the product catalog (Create, Read, Update, Delete).
* **User Management**: Overview of all registered users, ability to manage administrative privileges, or remove accounts.
* **Order Monitoring**: Full insight into all system orders with detailed delivery status tracking.
